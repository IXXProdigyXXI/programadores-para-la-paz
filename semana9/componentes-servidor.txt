Sistema operativo:
Es la base que administra el hardware del equipo y permite que el servidor funcione correctamente.
También controla recursos como memoria, procesos y red.

Node.js:
Es el entorno que permite ejecutar JavaScript en el servidor.
Gracias a Node.js podemos crear aplicaciones backend como esta práctica.

Aplicación server.js:
Es el archivo donde se define la lógica del servidor y sus rutas.
En este caso responde en / y en /estado.

Puerto:
Es el punto de entrada por donde el servidor recibe solicitudes.
El puerto 3000 indica dónde está escuchando la aplicación.

Logs:
Son mensajes o registros que informan lo que ocurre en el servidor.
Sirven para monitorear ejecución, detectar errores y revisar el comportamiento del servicio.
